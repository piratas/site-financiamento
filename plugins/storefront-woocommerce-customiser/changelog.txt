* New - Added option to add the product description on products in the loop.
* New - Added option to remove the product description on single products.
* New - Deeper integration with the Composite Products extension.

* Tweak - Only display homepage settings when customising the homepage and the Storefront homepage template is in use.
* Tweak - Only display shop settings on product archives.
